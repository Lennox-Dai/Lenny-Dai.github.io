  • 操作码的机器语言表示

    总体组成如下图：

    • 对于1-2字节，里面决定了这个操作是什么，以及是字操作还是字节操作，还有如果是双操作数的话，哪个用寄存器哪个用立即数

  • 8086指令系统

    1. 数据传送指令

      • MOV
      • PUSH
      • POP
      • XCHG
      • IN/OUT
      • XLAT(换码，要先有表，只能用AL和BX)
      • LEA
      • LDS/LES/LSS/LFS/LGS
      • LAHF
      • SAHF
      • PUSHF
      • POPF
      • CBW
      • CWD
      • CDQ

    2. 算术指令（除十进制调整指令外，其他指令均影响某些标志位）

      • 加法指令
      • 减法指令
      • 乘法指令
      • 除法指令（商在低位）
      • 十进制调整指令（注意除法）
        • DAA
        • DAS
        • AAA(非压缩)
        • AAS
        • AAM
        • AAD

    3. 逻辑指令

      • AND
      • OR
      • NOT
      • XOR
      • TEST
      • SHL\SHR
      • SAL\SAR
      • ROL\ROR
      • RCL\RCR

    4. 串处理指令

      • MOVS（串传送）数据段到附加段

        前置工作：

        1. 放原串首地址
        2. 放目的串首地址
        3. 放数据段长度到计数寄存器
        4. 建立方向标志

      • STOS(存入串)

        AX放到附加段

      • INS（串输入)

      • CMPS（串比较）俩串比

      • SCAS（串扫描）一个串一个数比

      • LODS（从串取）

    5. 控制转移指令

      • JMP
      • 条件转移（只能段转移）
      • LOOP
      • CALL
      • INT（存储器最低的1024字节，256种中断，低位偏移地址，高位段地址）

    6. 处理机控制指令
